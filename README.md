# Loan Approval Prediction

**Objective:** Predict whether a loan will be approved based on applicant information.

**Tools & Libraries:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn

**Steps:**
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA) with Pie Charts and Histogram
- Model Training using Random Forest
- Model Evaluation: Accuracy, Classification Report, Confusion Matrix

**Key Insights:**
- Credit History, Applicant Income, and Property Area are the most important factors affecting loan approval.

**Outcome:** Achieved X% accuracy on test data.
